# BonMenjar - Receptes Tradicionals Balears

## Descripció
BonMenjar és una aplicació web dedicada a la cuina tradicional de les Illes Balears. El projecte ofereix una plataforma interactiva per descobrir, explorar i compartir receptes autèntiques de la gastronomia balear.

## Característiques Principals
- Catàleg complet de receptes tradicionals balears
- Classificació per categories: entrants, plats principals, postres i begudes
- Sistema de valoracions i ressenyes per a cada recepta
- Integració amb mapes per localitzar restaurants
- Reproducció de vídeos demostratius
- Cercador de receptes
- Interfície responsive i moderna
- Suport per a narració de receptes mitjançant text-to-speech

## Tecnologies Utilitzades
- HTML5, CSS3, JavaScript
- Bootstrap 5
- APIs: Google Maps, YouTube
- Altres llibreries: AOS, Swiper, GLightbox
